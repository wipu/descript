Editing file, inlined content

~ $ $EDITOR "file1"
file1 line1
file1 line2
~ $ cat file1
file1 line1
file1 line2
Output asserted
~ $ $EDITOR "file1"
file1 line1
file1 line1, modified
file1 line2
file1 line3, added
~ $ cat file1
file1 line1, modified
file1 line2
file1 line3, added
Output asserted
~ $ $EDITOR "file1"
file1 line1, modified
file1 line2
file1 line3, added
~ $ cat file1
file1 line1, modified
file1 line3, added
Output asserted

Empty lines also work in editor:

~ $ $EDITOR "file1"
file1 line1, modified
file1 line3, added
empty line below:
empty line above
~ $ $EDITOR "file1"
file1 line1, modified
file1 line3, added
empty line below:
empty line above
empty line above and 2 below
~ $ $EDITOR "file1"
file1 line1, modified
file1 line3, added
empty line below:
empty line above and 2 below
empty line above and
none below
